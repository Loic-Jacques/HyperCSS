Le vaisseau

Le vaisseau de notre joueur.
Doté d'hyperpropulseurs, il file à toute allure dans l'espace inifini.

Les météores

Les météorites viennent du haut de l'ecran.
Elles se déplacent vers le bas de l'ecran.
Si elles touchent le vaisseau, le jeu est fini.

Les ennemis

Non implémenté, quand ils arriveront, ils tireront vers le vaisseau pour tenter de le détruire.
